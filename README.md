### ZSH Theme Setup
```
git clone --single-branch --branch next https://github.com/bhilburn/powerlevel9k.git ~/.powerlevel9k
```

### VIM Setup
```
curl -fLo ~/.vim/autoload/plug.vim --create-dirs \
    https://raw.githubusercontent.com/junegunn/vim-plug/master/plug.vim

:PlugInstall
```
